# 🍔 Food & Hospitality Management System

A full-stack web application that allows users to **order food and book rooms seamlessly**, with integrated payment functionality.

---

## 🚀 Features

* 🛒 Add food items to cart
* 🏨 Room booking system
* 💳 Razorpay payment integration
* 👤 User registration & login 
* 📦 Order storage in backend
* 📊 Dynamic cart updates
* 🔄 Real-time UI updates

---

## 🛠️ Tech Stack

### 🔹 Frontend

* HTML
* CSS
* JavaScript

### 🔹 Backend

* Java
* Spring Boot
* REST APIs

### 🔹 Database

* MySQL / H2 Database

---

## 📐 Architecture

This project follows a **3-tier architecture**:

### 🔹 1. Presentation Layer (Frontend)

* HTML, CSS, JavaScript
* Handles UI (Food menu, Cart, Room booking)

### 🔹 2. Application Layer (Backend)

* Spring Boot (Java)
* Handles:

  * User authentication
  * Order management
  * Room booking
  * Payment processing

### 🔹 3. Data Layer (Database)

* Stores:

  * Users
  * Orders
  * Rooms
  * Menu items

---

## 🔄 Application Flow

1. User interacts with UI (Frontend)
2. Frontend sends request to Backend (REST API)
3. Backend processes logic and interacts with Database
4. Response is sent back to UI
5. UI updates dynamically

---

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/chandramouli17/food-hospitality-management-system.git
```

2. Open project in IntelliJ / VS Code

3. Run Spring Boot application

4. Open browser:

```
http://localhost:8080
```
